# TrailingSlash

원형 : trail, slash 

의미  : 꼬리, 슬래시<span style="color:#FFBF00; font-weight:bold;">후행 슬래시</span>

발음 : 트레일링 슬래쉬

trailing slash란 URL끝에 있는 슬래시 입니다. trailing slash는 일반적으로 디렉토리임을 표시하기 위해 사용되어 지고, 만약 URL이 후행 슬래시 없이 끝난다면, 일반적으로 파일을 뜻합니다.
